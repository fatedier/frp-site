简单、高效的内网穿透工具

go get github.com/fatedier/frp

  • 易安装

    您可以根据操作系统平台直接下载二进制文件，解压后即可运行。

  • 高性能

    借助于 Go 语言的优势，使 frp 可以承载高并发的连接数，并且性能卓越。

  • 跨平台

    任何 Go 语言 支持的平台都可以运行 frp，包括 Windows、Mac、Linux 以及 ARM。frp 甚至可以运行在廉价的树莓派和路由器上。

  • 功能丰富

    frp 目前已经支持 tcp、udp、http、https 等协议，支持多用户，web 管理界面，并且在持续迭代更新中。
